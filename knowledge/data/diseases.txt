# 脐橙病虫害防治知识库

## 一、主要病害

### 1. 柑橘黄龙病（Huanglongbing, HLB）
**危害程度：极重（检疫性病害）**

- **病原**：韧皮部杆菌（Candidatus Liberibacter asiaticus）
- **症状**：
  - 初期：植株局部枝条叶片黄化，俗称"黄梢"。
  - 中期：出现斑驳黄化（黄绿相间，不均匀），果实小、畸形、色泽异常（"红鼻果"）。
  - 末期：全株黄化，树势衰退，最终死亡。
  - 根系：根系腐烂，树势极度衰弱。
- **传播途径**：由柑橘木虱（Diaphorina citri）传播，也可通过带病接穗传播。
- **防治措施**：
  - **检疫**：严禁从病区引进苗木和接穗，建立无病苗圃。
  - **防治木虱**：这是预防黄龙病的关键！用噻虫嗪（20%可溶液剂，1500-2000倍）、吡虫啉（10%可湿性粉剂，1000-2000倍）等防治木虱，在新梢萌发期喷药。
  - **病树处理**：发现病树立即砍除销毁，不可留用。
  - **补种无病苗**：补种经检疫合格的无病苗木。
  - **目前无治疗方法**，防控重点在于预防和切断传播途径。

### 2. 柑橘溃疡病（Citrus Canker）
**危害程度：重（检疫性病害）**

- **病原**：地毯草黄单胞杆菌柑橘致病变种（Xanthomonas citri subsp. citri）
- **症状**：叶片、枝梢、果实出现木栓化隆起病斑，初为针头状油渍状小点，扩大后中央凹陷呈火山口状，周围有黄色晕圈。
- **传播途径**：风雨、昆虫（潜叶蛾）、农事操作传播。
- **防治措施**：
  - 选用无溃疡病苗木。
  - 新梢萌发期喷施铜制剂（77%氢氧化铜可湿性粉剂800倍、20%噻菌铜悬浮剂500倍）保护。
  - 防治潜叶蛾，减少伤口。
  - 及时剪除病枝病叶并销毁。
  - 台风后及时喷药保护。
  - 套袋可有效减少果实溃疡病发生。

### 3. 柑橘炭疽病（Citrus Anthracnose）
- **病原**：胶孢炭疽菌（Colletotrichum gloeosporioides）
- **症状**：
  - 叶片：急性型（水渍状大斑）、慢性型（灰白色圆形病斑，边缘褐色）。
  - 枝梢：枝枯型（枝条由梢端向下枯死）。
  - 果实：干疤型（扁圆形褐色病斑）、泪痕型（果面泪痕状暗褐色条纹）、腐烂型。
- **防治措施**：
  - 加强栽培管理，提高树体抗性。
  - 喷施25%咪鲜胺乳油（1000-1500倍）、10%苯醚甲环唑水分散粒剂（2000-3000倍）、75%百菌清可湿性粉剂（600-800倍）。
  - 采果后喷药保护。

### 4. 脂点黄斑病（Greasy Spot）
- **病原**：柑橘褐色蒂腐病菌（Mycosphaerella horii）
- **症状**：叶背出现疱疹状油渍斑，后变黄，叶正面出现不规则黄斑，严重时导致落叶。
- **防治措施**：
  - 清除枯枝落叶，减少菌源。
  - 5-7月喷施铜制剂（石硫合剂、氢氧化铜等）或77%可杀得2000倍液。

### 5. 柑橘疮痂病（Citrus Scab）
- **病原**：痂囊腔菌（Elsinoe fawcettii）
- **症状**：叶片出现蜡黄色油渍状小点，后突起成木栓化瘤状，叶片扭曲畸形。果实出现木栓化粗糙斑。
- **发病条件**：春季多雨时期，气温20℃左右。
- **防治措施**：
  - 春芽萌发前喷波美0.5度石硫合剂。
  - 谢花2/3时喷70%甲基托布津1000倍液或75%百菌清600倍液。

### 6. 青霉病和绿霉病（储运期病害）
- **病原**：意大利青霉（Penicillium italicum）、指状青霉（P. digitatum）
- **症状**：储运期果实腐烂，分别呈蓝绿色和绿色霉状物。
- **防治措施**：
  - 采收时轻拿轻放，减少机械损伤。
  - 采后24小时内用咪鲜胺（500-1000mg/L）浸果处理。
  - 贮藏库消毒，控制温湿度。

## 二、主要虫害

### 1. 柑橘木虱（Diaphorina citri）
**危害程度：极重（黄龙病传播媒介）**

- **识别特征**：成虫体长2.7-3.0mm，灰褐色，头部蓝绿色，翅透明有褐色纹。若虫橙黄色，有白色蜡丝。
- **危害方式**：成虫和若虫刺吸新梢汁液；若虫排泄物可引起煤烟病；传播黄龙病。
- **发生规律**：一年发生8-12代，全年均可发生，嫩梢期危害严重。
- **防治方法**：
  - 统一放梢，减少零星抽梢，集中防治。
  - 新梢萌发1-2厘米时喷药：噻虫嗪（20%可溶液剂2000倍）、吡虫啉（10%可湿性粉剂1500倍）、螺虫乙酯（22.4%悬浮剂4000倍）。
  - 冬季清园，减少越冬虫口基数。

### 2. 柑橘红蜘蛛（Panonychus citri）
- **识别特征**：成螨红色，体长约0.3mm，椭圆形，有细毛。
- **危害方式**：刺吸叶片汁液，叶片出现灰白色失绿斑点（"花叶"），严重时落叶。
- **发生规律**：一年发生12-15代，春秋两季危害最重（4-5月、9-10月）。
- **防治方法**：
  - 虫口密度：每叶平均2-3头时防治。
  - 药剂：阿维菌素（1.8%乳油2000-3000倍）、哒螨灵（15%乳油2000倍）、螺螨酯（24%悬浮剂5000倍）、乙螨唑（11%悬浮剂5000倍）。
  - 注意轮换用药，防止产生抗药性。

### 3. 柑橘潜叶蛾（Phyllocnistis citrella）
- **俗名**：鬼画符。
- **危害方式**：幼虫潜入嫩叶表皮取食叶肉，留下弯曲隧道，叶片卷曲。潜叶蛾造成的伤口是溃疡病侵染的重要途径。
- **发生规律**：主要危害夏秋新梢（6-10月），新梢展叶后即可受害。
- **防治方法**：
  - 统一放梢，新梢长0.5-1厘米时喷药保护。
  - 药剂：甲维盐（1%乳油3000倍）、高效氯氟氰菊酯（2.5%乳油3000倍）、虫酰肼（20%悬浮剂1000倍）。

### 4. 褐圆蚧（Chrysomphalus aonidum）和矢尖蚧（Unaspis yanonensis）
- **危害方式**：刺吸叶片、枝条、果实汁液，引起黄化落叶，煤烟病，果实商品价值降低。
- **发生规律**：一年发生2-4代。
- **防治方法**：
  - 若虫孵化盛期（初夏、夏末）喷药：机油乳剂（99%乳油100-150倍）、噻嗪酮（25%可湿性粉剂1500-2000倍）、螺虫乙酯（22.4%悬浮剂4000倍）。
  - 冬季清园，喷施石硫合剂或机油乳剂。

### 5. 蚜虫（柑橘蚜，Toxoptera citricidus）
- **危害方式**：群集嫩梢、嫩叶刺吸汁液，叶片卷曲，分泌蜜露诱发煤烟病，并传播柑橘衰退病毒（CTV）。
- **防治方法**：
  - 嫩梢期喷施吡虫啉（10%可湿性粉剂1500-2000倍）、啶虫脒（3%乳油2000倍）等。

## 三、综合防治策略

### 农业防治
1. 选用抗病品种和无病苗木。
2. 加强栽培管理，增强树势，提高抗性。
3. 合理施肥，避免过量施氮。
4. 统一放梢，减少零星抽梢，降低病虫危害。
5. 冬季清园：清除枯枝落叶、翻土、喷石硫合剂（波美0.8-1度）。

### 物理防治
1. 色板诱杀：黄板诱杀蚜虫、白粉虱；蓝板诱杀蓟马。
2. 杀虫灯：诱杀金龟子、凤蝶等趋光性害虫。
3. 套袋：减少果实病虫害，降低农药使用量。

### 生物防治
1. 保护和利用天敌：捕食螨防治红蜘蛛，瓢虫、草蛉防治蚜虫、介壳虫。
2. 释放捕食螨（胡瓜钝绥螨）：防治柑橘红蜘蛛。
3. 应用Bt（苏云金杆菌）防治鳞翅目害虫。

### 化学防治原则
- 对症用药，精准施药。
- 轮换使用不同作用机理的农药。
- 严格遵守农药安全间隔期。
- 优先选用高效低毒低残留农药。
- 套袋前彻底用药，减少套袋后用药次数。

### 年度防治历
| 时期 | 主要防治对象 | 推荐药剂 |
|------|------------|---------|
| 冬季清园（12-2月）| 越冬虫卵、病菌 | 石硫合剂（波美0.8-1度）、机油乳剂 |
| 春梢期（2-4月）| 木虱、蚜虫、红蜘蛛、炭疽病、疮痂病 | 吡虫啉、阿维菌素、甲基托布津 |
| 花期（4月）| 木虱、红蜘蛛 | 注意不在花期喷杀虫剂（保护蜜蜂）|
| 幼果期（5-6月）| 木虱、潜叶蛾、炭疽病、溃疡病 | 噻虫嗪、氢氧化铜、咪鲜胺 |
| 夏梢期（6-8月）| 潜叶蛾、木虱、红蜘蛛、溃疡病 | 甲维盐、螺螨酯、噻菌铜 |
| 果实发育期（8-10月）| 介壳虫、炭疽病 | 机油乳剂、苯醚甲环唑 |
| 采收前30天 | 停止施药（严格执行安全间隔期）| — |
